# Coffee Quality Analysis

## Real-world data analysis project using SQL and Power BI to derive actionable business insights.

## 📌 Objective
Analyze coffee quality dataset to identify key factors affecting specialty-grade coffee and uncover high-performing countries and processing methods.

## 📊 Dataset
- Source: Coffee Quality Institute (CQI)
- Records: 200+ samples
- Attributes: 24 columns (Aroma, Flavor, Defects, Processing Method, Country, etc.)

## 🛠 Tools Used
- SQL (MySQL for data analysis)
- Power BI
- Excel (Data Cleaning)

## 🔍 Key Analysis
- Identified top-performing countries based on average quality scores
- Analyzed impact of processing methods on coffee quality
- Filtered high-quality coffee samples (>84 points)
- Studied effect of defects on quality
- Found strong correlation between Aroma and overall quality

## 📈 Dashboard Features
- KPI card showing average quality score
- Top 5 countries visualization
- Processing method comparison
- Defects impact analysis
- Aroma vs Quality scatter plot
- Interactive slicers (Country, Processing Method)

## 🚀 Key Insights
- Ethiopia and Tanzania are top-performing countries
- Double Anaerobic Washed method shows highest quality scores
- Aroma has strong positive correlation with coffee quality
- High-quality coffee (>84) represents a smaller segment of dataset
